## 挂载


{% highlight Bash %}
[root@template ~]# mkdir /storage
[root@template ~]# mount /dev/vdb1 /storage
{% endhighlight %}


然后要所挂载信息保存到/etc/fstab中，这样下载重启后就会自动挂载了
{% highlight Bash %}
[root@template ~]# vi /etc/fstab
{% endhighlight %}
按G到最后一行，再按o新增一行，输入“/dev/vdb1               /storage                ext4    defaults        1 2 ”
